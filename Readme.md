# Quark2Baidu 跨网盘秒传转存工具

> 一个基于 Python 的命令行工具，支持从夸克网盘选择文件并通过秒传技术转存到百度网盘，无需重复下载上传大文件。

## 功能特点

- 🚀 **秒传技术**：利用百度网盘秒传功能，避免重复上传
- 🖥️ **交互式界面**：支持 curses 图形界面选择文件（Windows/Linux/macOS）
- 📁 **批量操作**：支持多文件同时转存
- 🔒 **Cookie 认证**：使用浏览器 Cookie 进行网盘认证
- 📊 **实时进度**：显示转存状态和结果
- 🛡️ **安全可靠**：本地运行，不收集用户数据

## 系统要求

- Python 3.6+
- 操作系统：Windows / Linux / macOS

## 安装依赖

```bash
pip install requests httpx
```

**Windows 用户额外安装**（如需使用图形界面）：
```bash
pip install windows-curses
```

## 使用说明

### 1. 获取 Cookie

#### 夸克网盘 Cookie
1. 登录 [夸克网盘](https://pan.quark.cn)
2. 按 F12 打开开发者工具
3. 复制整个 `document.cookie` 内容

#### 百度网盘 Cookie
1. 登录 [百度网盘](https://pan.baidu.com)
2. 按 F12 打开开发者工具  
3. 复制整个 `document.cookie` 内容

### 2. 运行程序

```bash
python quark_to_baidu.py
```

### 3. 操作步骤

1. **输入夸克 Cookie** - 粘贴夸克网盘 Cookie
2. **选择文件** - 在交互界面中选择要转存的文件
   - ↑/↓：移动光标
   - Enter：进入文件夹
   - Space：选择/取消选择文件
   - Backspace：返回上级目录
   - s：确认选择并开始转存
   - q：退出
3. **输入百度 Cookie** - 粘贴百度网盘 Cookie
4. **等待转存完成** - 程序自动执行秒传转存

## 技术原理

1. **文件信息获取**：通过夸克网盘 API 获取文件下载链接和元数据
2. **校验数据计算**：
   - 计算文件前 256KB 的 MD5（slice_md5）
   - 根据特定算法计算校验分片偏移量
   - 下载校验分片并 Base64 编码
3. **秒传转存**：使用百度网盘秒传 API 完成转存

## 注意事项

⚠️ **重要提醒**：

- 确保 Cookie 有效且未过期
- 百度网盘账户需要有足够存储空间
- 秒传功能依赖百度服务器已有相同文件
- 大文件处理需要稳定网络连接
- 请遵守相关服务条款，合理使用

## 故障排除

### 常见问题

1. **Cookie 失效**
   - 重新获取最新 Cookie
   - 确保登录状态有效

2. **转存失败**
   - 检查网络连接
   - 确认百度网盘账户正常
   - 文件可能被百度网盘屏蔽

3. **界面显示异常**
   - 尝试使用简易文本模式
   - Windows 用户安装 `windows-curses`
