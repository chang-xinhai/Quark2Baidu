# Quark2Baidu 跨网盘秒传转存工具

> 一个基于 Python 的命令行工具，支持从夸克网盘选择文件并通过秒传技术转存到百度网盘，无需重复下载上传大文件。

## 功能特点

- 🚀 **秒传技术**：利用百度网盘秒传功能，避免重复上传
- 🖥️  **TUI交互界面**：基于prompt_toolkit的现代化终端界面
- 📁 **智能目录操作**：支持递归选择整个文件夹，保持目录结构转存
- 📁 **批量操作**：支持多文件同时转存
- 🔒 **配置持久化**：Cookie 和设置自动保存，下次使用无需重复输入
- 📊 **实时进度**：进度条显示转存状态和结果
- 🛡️ **安全可靠**：本地运行，不收集用户数据

## 系统要求

- Python 3.6+
- 操作系统：Windows / Linux / macOS

pip install httpx prompt_toolkit tqdm

## 快速开始

### 1. 获取 Cookie

#### **方法 1：使用 Cookie-Editor 插件（推荐）**

**Chrome/Edge 用户**：
1. 安装 [Cookie-Editor](https://chromewebstore.google.com/detail/cookie-editor/hlkenndednhfkekhgcdicdfddnkalmdm ) 插件
2. 登录 [夸克网盘](https://pan.quark.cn ) 和 [百度网盘](https://pan.baidu.com )
3. 点击插件图标 → 点击 **Export** → 选择 **Header String** 格式复制

**Firefox/Safari 用户**：

1. 搜索并安装类似的 Cookie 导出插件
2. 登录相应网盘
3. 使用插件的导出功能复制完整 Cookie 字符串

#### **方法 2：从 Application 面板获取**

1. 按 **F12** 打开开发者工具
2. 切换到 **Application** 标签 → 左侧选择 **Cookies**
3. 在右侧表格中，**选中所有 Cookie**（包括 HttpOnly 的）
4. 右键 → **Copy → Copy All as Cookie String**
5. 完整粘贴到程序配置中

#### **方法 3：从 Network 面板获取（备用）**

1. 登录夸克网盘/百度网盘
2. 按 **F12** 打开开发者工具 → 切换到 **Network** 标签
3. 刷新页面（**F5**）或点击一个文件夹
4. 在请求列表中，点击任意 `sort` 或 `list` 请求
5. 右侧找到 **Request Headers** → **Cookie** 字段
6. 右键 Cookie 值 → **Copy value**
7. 完整粘贴到程序配置中

### 2. 运行程序并配置

```bash
python Q2B.py
```

首次运行会自动进入配置向导：
1. 粘贴夸克网盘完整 Cookie
2. 粘贴百度网盘完整 Cookie
3. 设置百度网盘保存路径（默认：`/Q2B/`）
4. 设置并发线程数（默认：`3`）

配置会自动保存到 `config.json`，下次运行无需重复输入。

### 3. 选择文件并转存

1. **启动文件浏览器**：程序自动加载夸克网盘文件列表
2. **浏览和选择**：
   - **↑/↓**：移动光标选择文件/文件夹
   - **Enter**：进入文件夹
   - **Backspace**：返回上级目录
   - **Space**：选择/取消选择文件
   - **r**：递归选择整个文件夹（显示 🔁 标记）
   - **s**：确认选择并开始转存
   - **Ctrl+C**：退出程序
3. **等待转存完成**：程序自动执行秒传转存，进度条实时显示状态

**选择状态说明**：

- `[ ]`：未选择
- `[✓]`：已选择文件
- `[D]`：已选择文件夹（未递归）
- `🔁`：已递归选择文件夹（包含所有子文件）

## 配置文件说明

`config.json` 包含以下设置项：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `quark_cookie` | 夸克网盘完整 Cookie | `""` |
| `baidu_cookie` | 百度网盘完整 Cookie | `""` |
| `target_path` | 百度网盘保存路径（必须以 `/` 开头和结尾） | `/Q2B/` |
| `concurrency` | 并发线程数（建议 3-10） | `3` |
| `chunk_size` | 验证分片大小（Byte）（正常不需要更改） | `262144` |
| `verify_ssl` | 是否验证 SSL 证书 | `true` |

## 故障排除

### 常见问题与解决方案

1. **Cookie 失效**
   - **现象**：提示"Cookie 无效或已过期"
   - **解决**：重新登录网盘并刷新 Cookie，建议使用插件一键导出

2. **SSL 证书验证失败**
   - **现象**：连接错误 or SSL 相关异常
   - **解决**：在配置向导中设置 `verify_ssl` 为 `false`

3. **转存失败**
   - **现象**：秒传失败
   - **原因**：百度网盘服务器无此文件缓存，或文件被屏蔽
   - **解决**：无法解决，这是百度服务端限制

4. **界面显示异常**
   - **现象**：菜单错位或颜色异常
   - **解决**：确保终端支持 UTF-8 和 ANSI 颜色代码

5. **递归选择无反应**
   - **现象**：按 `r` 键后文件夹未标记 🔁
   - **解决**：递归需要等待扫描完成，大文件夹可能需要几秒
